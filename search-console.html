Search Console登録手順

Googleにサイト構成を伝えるための登録手順です。公開URLは GitHub Pages のURLを使います。

プロパティ追加所有権確認サイトマップ送信送信後の確認

1. プロパティを追加する

Search Consoleで「プロパティを追加」を開き、URLプレフィックスに次のURLを入力します。

https://atsushisora.github.io/ichigeki-web/

独自ドメインを使っていない間は、GitHub PagesのこのURLで登録します。

2. 所有権を確認する

GitHub Pagesでは、HTMLファイルアップロードまたはHTMLタグ確認が使いやすい方法です。

  • HTMLファイル確認Googleが指定した確認ファイルをサイトのルートに置きます。ファイル名と中身をそのまま使う必要があります。
  • HTMLタグ確認Googleが出すmetaタグをトップページのhead内へ追加します。タグが分かったら、その時点でコードに入れます。

確認コードはアカウント固有なので、このページには直接書いていません。Search Console画面に表示されたコードを使います。

3. サイトマップを送信する

所有権確認が終わったら、Search Consoleの「サイトマップ」から次のURLを送信します。

https://atsushisora.github.io/ichigeki-web/sitemap.xml

このサイトでは、トップ、各シミュレーター、ランキング、使い方、ポリシー系ページをサイトマップに含めています。

4. 送信後に見るところ

  • サイトマップの取得が「成功」になっているか
  • インデックス登録済みページが増えているか
  • ページに重大なエラーが出ていないか
  • スマホでの表示に問題がないか

反映には時間がかかります。すぐに検索結果へ表示されない場合でも、サイトマップが成功していれば数日単位で確認します。
